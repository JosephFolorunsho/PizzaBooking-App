

# Pizza Booking Website

A simple web-based pizza booking application built with vanilla JavaScript.  
This project focuses on core web fundamentals such as DOM manipulation, event handling, and application logic without the use of frameworks.

## Overview
The application allows users to browse pizza options and complete a booking flow using plain JavaScript.  
It was created to demonstrate an understanding of core web concepts and how booking logic can be implemented without relying on libraries or frameworks.

## Features
- Pizza listing and selection
- Booking / ordering flow
- Form handling and validation
- Dynamic UI updates using the DOM
- Basic state handling in JavaScript

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript

## Project Focus
This project highlights:
- Strong JavaScript fundamentals
- DOM manipulation and event-driven logic
- Clean separation of concerns
- Building interactive web applications without frameworks
